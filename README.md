# library

Desktop Java Swing Library Management application.

## Run locally

```bash
javac LibraryGUI.java
java LibraryGUI
```

## Note about Render

Render is built for web/background services. A Java Swing desktop GUI cannot be hosted as an interactive web app on Render directly.

If you still want to upload this repository to Render, keep the repo name as `library` and deploy it only for build/demo purposes (not GUI runtime).
